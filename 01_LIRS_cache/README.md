# LIRS-cache
---
## How to build the Lirs-cache: 

* mkdir build
* cd build
* cmake ../Lirs
* make 
---
## Compiler settings:
```
-Wall -Wextra -Wpedantic -O2 -std=c++17
```
In a folder 'build' a file 'lirs' will appear executing a program
